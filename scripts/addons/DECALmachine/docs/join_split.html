  • Join and Split

Join

The Join tool is used to simplify and organize your scene, by creating a single Decal object from a selection of Decals, in a non-destructive manner.

Attention

It is adviced to never use Blender's native Join tool to join Decals, as it's very destructive.

Selection

Any number of Decal objects

Split

The Split tool then is used to split previously joined Decals apart again.

Attention

The Split tool only works for Decals joined using DECALmachine's Join tool, not for Decals joined via Blender's native Join.

Selection

Any number of previously joined Decal objects

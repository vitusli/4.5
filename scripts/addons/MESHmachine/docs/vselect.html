Vselect

The VSelect tool, found in the Select sub-menu, can be used to quickly select geometry based on vertex group membership.

Also see the Select tool, for a more convenient way to access VSelect.

Selection

If nothing is selected, all vertex groups of the mesh will be cyclced through.
Otherwise only the vertex groups the selected verts, edges or faces belong to will be availble to choose from..
